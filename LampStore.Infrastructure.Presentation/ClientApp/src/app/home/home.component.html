LampWebStore demo project

This is my WebApi demo project.

The back-end was made by means of asp.core with the onion architecture. It use EF.core to communicate with data base,
the Fluent Validation library to perform input data checking,
the AutoMapper library to perform DTO conversions,
the Angular framework to build presentation SPA as a front-end.

This demo consists of the Lamps entities with all CRUD operations are allowed on them.
Also it allows produce comparisons between a couples of the lamps.
